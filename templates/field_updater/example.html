Field Updater Examples

simple use

Provide a submit url and a single key word argument
Click the value to show an input, edit it and submit A POST request will be sent to the url with body form encoded key=value

{% verbatim %} {% url 'cache_submit' as submit_url %}
{% field_updater submit_url=submit_url key='value' %} {% endverbatim %}
{% url 'cache_submit' as submit_url %} {% field_updater submit_url=submit_url key='value' %}
